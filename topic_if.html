Задача №1: Фейс-контроль

Теорія: if перевіряє умову. Якщо вона істинна, виконується код з відступом (4 пробіли).

Запитай вік. Якщо він 18 або більше, виведи "Доступ дозволено".

Вік: > 20
Доступ дозволено
age = int(input("Вік: "))
if age >= 18:
print("Доступ дозволено")

Задача №2: Пароль

Теорія: Для перевірки рівності використовуй подвійне дорівнює ==.

Запитай пароль. Якщо він дорівнює "1234", виведи "Вхід виконано".

p = input("Пароль: ")
if p == "1234":
print("Вхід виконано")

Задача №5: Парне чи ні?

Теорія: else спрацьовує, якщо умова в if виявилася хибною.

Запитай число. Якщо воно ділиться на 2 без остачі (n % 2 == 0), виведи "Парне", інакше — "Непарне".

Число: > 7
Непарне
n = int(input("Число: "))
if n % 2 == 0:
print("Парне")
else:
print("Непарне")

Задача №9: Оцінка школи

Теорія: elif (скорочено від else if) дозволяє перевірити кілька умов по черзі.

Запитай бал (1-12). Якщо 10-12 — "Високий", якщо 7-9 — "Достатній", інакше — "Треба вчитися".

mark = int(input("Бал: "))
if mark >= 10:
print("Високий")
elif mark >= 7:
print("Достатній")
else:
print("Треба вчитися")
